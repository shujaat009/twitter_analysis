Real-Time Twitter Analysis

This project is focused on real-time Twitter analysis using Power BI. Our goal is to provide insights into trending topics, sentiment analysis, and other key metrics from Twitter data.

Objectives

  • Analyze real-time Twitter data
  • Visualize key metrics using Power BI
  • Provide actionable insights

Team

This project is developed by Md Shujaat Hussain and the team at Nullclass Pvt Ltd.

Technologies Used

We have used Power BI, Python, MySQL, and other data analysis tools.
